Hướng Dẫn Cài Linux (Ubuntu) trên VMware

AUG 19, 2017

Ubuntu là một hệ điều hành máy tính dựa trên Debian GNU/Linux, một bản phân phối Linux thông dụng. Mục đích của Ubuntu bao gồm việc cung cấp một hệ điều hành ổn định, cập nhật cho người dùng thường, và tập trung vào sự tiện dụng và dễ dàng cài đặt

Ubuntu là phần mềm mã nguồn mở tự do, có nghĩa là người dùng được tự do chạy, sao chép, phân phối, nghiên cứu, thay đổi và cải tiến phần mềm theo điều khoản của giấy phép GNU GPL

Phần cứng tối thiểu của máy tính bạn

  • CPU: Core Pentium trở lên (dòng Intel Pentium G4400 trở lên là dư sức rồi, Dual-Core cũng được mà hơi yếu thôi)
  • RAM: 4GB
  • Ổ Cứng: Dư khoảng 30GB trên ổ cứng (khuyên nên xài SSD thì tốc độ nhanh hơn rất nhiều)

Các công cụ để cài Linux trên Vmware

  • Ubuntu: Download
  • VMware Player: Download

1. Cài đặt VMware

Sau khi tải VMware Player chạy file VMware-player-12.5.7-5813279.exe để bắt đầu cài đặt. Sau đó bạn nhấn next cho tới khi cài đặt xong. Lưu ý khi mở lên VMware lên bạn phải đăng nhập email của bạn để kích hoạt

2. Tạo máy ảo Linux

2.1 Mở VMware Player lên chọn Create a New Virtual Machine

2.2 Chọn Installer disc image file (iso) và nhấn Browse...

2.3 Chọn file Ubuntu bạn đã tải về ubuntu-17.04-desktop-amd64.iso và nhấn Open

2.4 Nhấn Next

2.5 Điền Tên, Tên đăng nhập và mật khẩu rồi nhấn Next. Vui lòng nhớ mật khẩu vì đây là mật khẩu để bạn đăng nhập vào máy tính

2.6 Đổi tên theo ý của bạn và nhấn Next. Giữ nguyên đường dẫn Location vì đây là file Configuration của máy, bạn không cần đụng nếu không biết đây là gì

2.7 Tick vào Store virtual disk as a single file và chọn dung lượng cho máy ảo của bạn và nhấn Next

2.8 Nhấn Customize Hardware...

2.9 Chọn dung lượng Ram cho máy ảo của bạn. Máy tình 8GB nên chọn 4GB cho mạnh (Ubuntu chỉ cần 2GB hoặc trên là đủ chiến rồi)

2.10 Nhấp vào Processors và chọn số nhân CPU tùy thuộc theo chip của bạn

2.11 Dành cho ai xài mạng LAN, bạn nào xài Wifi có thể bỏ qua bước này. Chọn vào Network Adapter và nhấn vào Bridged: Connected... , tick vào Replicate physical network... rồi nhấn Close

2.12 Chọn Finish

3. Chạy và cài đặt Linux

3.1 Chọn Play virtual machine

3.2 Khi nhấn Play máy ảo sẽ Boot và tự động cài đặt, khoảng vài giây sau sẽ có Pop-up Software Updates, bạn nhấn Download and Install

3.3 Máy trong quá trình cài đặt

3.4 Khi cài đặt xong máy sẽ khởi động lại và vào màn hình Lock Screen. Nhập mật khẩu bạn đã tạo ở bước 2.5

Thế là xong bây giờ bạn có thể dùng như một chiêc máy bình thường
