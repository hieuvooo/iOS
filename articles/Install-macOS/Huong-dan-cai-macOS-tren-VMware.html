Hướng Dẫn Cài macOS High Sierra Trên VMware

Aug 31, 2017

macOS một hệ điều hành độc quyền, được phát triển và phân phối bởi Apple, được cài đặt sẵn trên các máy tính Macintosh. Để xài macOS thông thường bạn phải mua máy tính của Apple như Macbook, iMac hay Mac Pro... Nhưng đối với điều kiện kinh tế của từng người thì không phải ai cũng có thể sắm cho mình một con Macbook hay iMac được. Bài này sẽ hướng dẫn cho tất cả người dùng phổ thông đang ngồi trên một chiếc PC hoặc Laptop chạy Windows có thể cài macOS trên máy mình

Để chạy macOS trên máy tính thông thường, bạn phải cần VMware hoặc cài thẳn lên máy tính, cách này hơi tốn thời gian và rườm rà hơn chạy VMware nên mình chỉ cài trên VMware thôi

Phần cứng tối thiểu của máy tính bạn

  • CPU: Core i3 trở lên (dòng i3-6000 trở lên là dư sức rồi)
  • RAM: 8GB trở lên (6GB cũng được, bạn nào dưới 6GB thì chạy ra chợ mua thêm vài gb ram nhé)
  • GPU: GT 1030 trở lên (không có cũng được, có điều hình hơi giật thôi)
  • Ổ Cứng: Dư khoảng 100GB trên ổ cứng (khuyên nên xài SSD thì tốc độ nhanh hơn rất nhiều)

Các công cụ để cài macOS trên Vmware

  • macOS High Sierra: Download
  • Tools: Download
  • VMware Player: Download

1. Cài đặt VMware

Sau khi tải VMware Player chạy file VMware-player-12.5.7-5813279.exe để bắt đầu cài đặt. Sau đó bạn nhấn next cho tới khi cài đặt xong. Lưu ý khi mở lên VMware lên bạn phải đăng nhập email của bạn để kích hoạt

2. Cài Patch VMware để hoạt động trên macOS

Sau khi tải Tools về bạn giải nén và vào thư mục Unlock VM tìm file win-install.cmd click chuột phải chọn Run as administrator và cài đặt đợi cho đến khi CMD chạy xong sẽ tự thoát

3. Giải nén macOS

Sau khi tải macOS về. Tạo một thư mục và đặt tên nó bất kì bạn muốn và copy file macOS High Sierra Beta.rar vào thư mục bạn vừa tạo rồi giải nén tại thư mục đó sẽ có được file macOS High Sierra Beta.vmdk

4. Tạo máy ảo macOS

4.1 Mở VMware Player lên chọn Create a New Virtual Machine

4.2 Chọn I will install the operating system later và chọn next

4.3 Chọn Apple Mac OS X version là macOS 10.12 và chọn next

4.4 Đổi tên tùy theo ý bạn và chọn next. Giữ nguyên đường dẫn Location vì đây là file Configuration của máy

4.5 Tick vào Store virtual disk as a single file rồi chọn next. Lưu ý không cần đụng vào phần ổ cứng vì lát ta cũng bỏ nó đi

4.6 Chọn Finish để hoàn thành

4.7 Chọn Edit virtual machine settings

4.8 Chọn mức ram cho macOS. Mình lấy tổng dung lương ram chia đôi và cho thêm nó 1Gb tổng cộng là 5GB

4.9 Sau khi chọn ram phù hợp với bạn, nhấp vào Processors và chọn số nhân CPU. Ví dụ máy mình core i5 4 nhân thì mình chọn 4

4.10 Chọn xong số nhân thì bạn nhấn vào Hard Disk (SATA) rồi chọn Remove

4.11 Chọn Add

4.12 Chọn Hard Disk và nhấn Next

4.13 Chọn SATA (Recommended) và nhấn Next

4.14 Chọn Use an existing virtual disk và nhấn Next

4.15 Chọn vào Browse

4.16 Mở tự mục bạn đã giãi nén macOS ở bước 3 và chọn File macOS High Sierra Beta.vmdk và nhấn Open

4.17 Chọn Finish để hoàn thành

4.18 Dành cho ai xài mạng LAN, bạn nào xài Wifi có thể bỏ qua bước này. Chọn vào Network Adapter và nhấn vào Bridged: Connected... , tick vào Replicate physical network... rồi nhấn OK

4.19 Xem đường dẫn Location ở bước 4.4 rồi mở đường dẫn đó. Trong thư mục sẽ có file macOS 10.12.vmx bạn nhấp chuột phải chọn Open with Notepad kéo xuống dưới cùng và thêm dòng smc.version = "0" rồi nhấn Ctrl + S để lưu lại

5. Chạy và cài đặt macOS

5.1 Chọn Play virtual machine

5.2 Chọn Play virtual machine

5.3 Đợi khoảng 5p tùy thuộc vào ổ cứng của bạn. Mình để macOS trên SSD nên mất khoảng hơn 2p để boot tới Setup

5.4 Sau khi chạy xong sẽ ra Setup máy, bạn cứ việc setup bình thường giống như trên iPhone nên bước này mình sẽ không hướng dẫn

5.5 Trong khoảng thời gian bạn đang Setup thì VMware sẽ báo khung màu vàng ở dưới bạn chọn vào Install Tools rồi tiếp tục Setup nhé

5.6 Sau khi chạy Setup xong thì sẽ ra màn hình chính, đến bước này vẫn chưa xong đâu, bạn còn phải chạy Tools để điều chỉnh hiệu năng và sửa một số files để macOS của bạn như một máy thật. Chạy VMware Tools để tiếp tục

5.7 Chọn Install VMware Tools

5.8 Chọn Continue

5.9 Chọn Install

5.10 Chọn Continue Installation

5.11 Nhập mật khẩu lúc bạn cài lúc Setup máy

5.12 Khởi động lại máy chọn Restart

5.13 Sau khi khởi động lại, máy sẽ báo System Extension Blocked, bạn cứ nhấn OK

5.14 Mở System Preferences dưới thanh Dock, bạn nào không biết System Preferences thì nhìn dưới hình nhé

5.15 Chọn Security & Privacy

5.16 Nhấp vào biểu tượng hình ổ khóa rồi nhập mật khẩu của bạn

5.17 Nhập mật khẩu xong lúc này hình ổ khóa sẽ được mở, lúc này nhấp Allow

5.18 Sau khi Allow thì nhấp vào biểu tượng trái Táo góc trái trên cùng và nhấn Restart...

5.19 Sau khi Restart máy sẽ khởi động lại và vào màn hình chính

6. Fix lỗi lặt vặt trên macOS

6.1 Fix lỗi About This Mac không hiển thị đúng. Mở thư mục mà bạn giải nén Tools ở phần 2. Trong thư mục sẽ có 3 Files, bạn tiềm file VM Fix.txt và mở lên, Copy phần từ phần board-id... cho tới efi.nvram...

6.2 Tới đường dẫn Location ở bước 4.4 của bạn, mở file macOS 10.12.vmx dưới NotePad

6.3 Nhấn Ctrl + F để mở bản tiềm kiếm, ở Find what gõ board-id và Direction chọn Up

6.4 Kiếm được board-id.reflectHost thì bạn đổi từ "TRUE" thành "FALSE", nhớ viết hoa. Sau khi copy phần board-id ở bước 6.1 thì bạn nhấn Enter ngay chổ bạn vừa sửa thành "FALSE" để xuống dòng và paste vào, sau đó nhấn Ctrl + S để lưu lại là xong. Bây giờ  trong phần About This Mac sẽ có đầy đủ thông tin

Xong. Thế là bạn có một macOS hoàn chỉnh chạy trên VMware

Một số Tips hay

  • Nhấn Ctrl + Alt để thoát ra trỏ chuột chính, nhấn tiếp Ctrl + Alt + Enter để thu nhỏ VMware
  • Nút Windows là nút Command. Nếu bạn muốn Copy trong macOS thì nhấn Windows + C và Windows + V là Paste
